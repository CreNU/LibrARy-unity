# Contributing Rules

## Commit Log
+ 구체적으로 작성해 주세요.

## Coding Style
### 들여쓰기
+ 들여쓰기는 공백 4칸을 사용합니다.

### 변수
+ **파스칼** 표기법을 사용합니다.
+ 첫 글자를 대문자로 표기합니다.
+ bool 변수는 is로 시작합니다.

### 함수
+ 파스칼 표기법을 사용합니다.

### 조건문
- 조건문에서 NOT은 (조건) == false 이렇게 합니다.
```
if (ilililii == false) // (O)

if (!ilililii) // (X)
```

- 변수와 상수를 비교할 땐 **변수를 왼쪽**, **상수를 오른쪽**에 써주세요.
```
if (a < 3)//(O)

if (3 > a)//(X)
```

### 중괄호
+ 중괄호는 내려서, 한줄이더라도 씁니다.
```
if (NoBrace)
{
    BraceWillBeSad();
}
```
